Niveaux de classification

Cette page présente les différents niveaux de classification utilisés dans le système ShadowComm.

Liste des niveaux

Niveau S

Le niveau S est le niveau le plus sensible.

  • Accès très limité
  • Informations très secrètes
  • Réservé à peu d'agents

Niveau A

Le niveau A donne accès à des informations importantes.

  • Accès contrôlé
  • Données importantes
  • Utilisé par plusieurs agents

Niveau C

Le niveau C est le niveau le plus simple.

  • Accès plus large
  • Informations moins sensibles
  • Utilisé pour les besoins courants
